III. CẤU TRÚC HỆ THỐNG

1. Mô hình kiến trúc:
[x] MVC (Model-View-Controller) - Bắt buộc
[ ] Microservices (Khuyến khích nếu đủ khả năng)

Ghi chú: Kiến trúc Full-stack Next.js với App Router + API Routes

2. Thiết kế Database (Tối thiểu 8 bảng):
Liệt kê tên các bảng (Tables/Collections) chính dự kiến:

1. users                          5. nft_listings
2. posts                          6. transactions  
3. comments                       7. notifications
4. nfts                           8. messages

Ghi chú bổ sung: 
- users: Thông tin người dùng (wallet address, role, profile)
- posts: Bài đăng của người dùng
- comments: Bình luận trên bài đăng
- nfts: Metadata NFT đã mint
- nft_listings: NFT đang được bán trên marketplace
- transactions: Lịch sử giao dịch mua/bán NFT
- notifications: Thông báo cho người dùng
- messages: Tin nhắn giữa các người dùng

3. Phân hệ chức năng (Tối thiểu 10 chức năng/site):

●  Site Admin:
   ○ Dashboard thống kê (số user, post, NFT, giao dịch)
   ○ Quản lý người dùng (xem, suspend, delete)
   ○ Quản lý bài đăng (duyệt, xóa vi phạm)
   ○ Quản lý báo cáo vi phạm
   ○ Cài đặt hệ thống

●  Site Public/User:
   ○ Đăng nhập/Đăng ký bằng Wallet (Dynamic Labs)
   ○ Xem feed bài đăng (Home)
   ○ Tạo bài đăng mới
   ○ Mint NFT từ bài đăng
   ○ Marketplace (mua/bán NFT)
   ○ Xem profile người dùng
   ○ Nhắn tin giữa người dùng
   ○ Bình luận, tương tác bài đăng
   ○ Tìm kiếm (Discover)
   ○ Quản lý thông báo
   ○ Cài đặt profile cá nhân

4. Tính năng nâng cao (Điểm cộng):
[x] Triển khai lên Hosting/VPS thực tế (Có domain)
    → Vercel: metabased.vercel.app
[x] Xây dựng RESTful API (Tối thiểu 10 endpoints)
    → API Routes:
       1. POST /api/auth/verify - Xác thực JWT
       2. GET  /api/auth/role - Lấy role user
       3. GET  /api/posts - Lấy danh sách bài đăng
       4. POST /api/posts - Tạo bài đăng mới
       5. GET  /api/posts/[postId] - Chi tiết bài đăng
       6. DELETE /api/posts/[postId] - Xóa bài đăng
       7. POST /api/comments - Tạo bình luận
       8. POST /api/nft/mint - Mint NFT
       9. GET  /api/nft/metadata - Lấy metadata NFT
       10. POST /api/marketplace/list - Đăng bán NFT
       11. POST /api/marketplace/buy - Mua NFT
       12. GET  /api/admin/users - Quản lý users (Admin)
       13. GET  /api/admin/stats - Thống kê (Admin)
[x] Tích hợp thanh toán online (Momo/VNPay/Stripe sandbox)
    → Tích hợp thanh toán bằng Cryptocurrency trên Base Sepolia (ETH testnet)
    → Smart Contracts: NFT721, NFT1155, Marketplace
